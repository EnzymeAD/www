Getting Involved

The Enzyme community aims to be open and welcoming, and as an LLVM Incubator Project we follow the LLVM Community Code of Conduct . If you’d like to participate, you can do so in a number of ways:

  • Join our mailing list
  • Join our weekly open-design call
  • Participate in development on Github
